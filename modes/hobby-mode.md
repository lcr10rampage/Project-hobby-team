# Hobby Mode

Hobby Mode helps the user understand a hobby, decide how involved to become, and plan gear, skills,
cost, safety, time, and (when relevant) a physical setup.

## Focus
Understanding the hobby · desired involvement · required gear/equipment · skills to learn ·
practice and progression · costs · time commitment · safety · storage · maintenance · seasonal
considerations · trips/events/experiences · optional physical setups or builds.

## Default specialists (activate)
- Project Coordinator
- Designer
- Functionality Specialist
- Cost Specialist
- Risk Specialist
- Time Specialist
- Materials Specialist
- Tools and Skills Specialist
- Research and Compatibility Specialist
- Documentation Specialist

## Optional specialists (activate only when triggered)
- Measurement Specialist
- Sketch Specialist
- 3D Model Specialist
- Build Sequence Specialist
- Testing and Verification Specialist

**Triggers** — activate the optional set when the hobby includes any of: a physical setup, a
workspace, a vehicle, a custom storage solution, equipment mounting, a structure, a build, a
detailed layout, custom electronics, or a restoration.

Record the reason whenever an optional specialist is activated or a default one is deactivated.

## Workflow
```
Hobby classification
  → Goals and desired involvement
  → Required gear and skills
  → Cost options
  → Safety and risk plan
  → Time and learning plan
  → Setup or layout planning (when needed)
  → Seasonal or maintenance planning
  → Final hobby guide
```

## Stages
CLASSIFICATION → GOAL_DEFINITION → RESEARCH → GEAR_PLANNING → SKILL_PLANNING → COST_REVIEW →
RISK_REVIEW → SCHEDULE_PLANNING → READY_TO_START → ACTIVE → REVIEW → EXPANSION.
